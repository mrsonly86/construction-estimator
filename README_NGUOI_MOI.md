## 🚨 GẶP LỖI? XEM NGAY!

### ❌ "Python is not recognized"
```
🔧 KHẮC PHỤC:
1. Gỡ Python cũ (Control Panel → Programs)  
2. Cài lại Python từ python.org
3. ✅ NHỚ TÍCH "Add Python to PATH"
4. Restart máy tính
```

### ❌ "No module named 'pandas'"
```
🔧 KHẮC PHỤC:
1. Mở Command Prompt (cmd)
2. Gõ: pip install pandas openpyxl reportlab matplotlib
3. Chờ cài đặt xong
```

### ❌ Không chạy được file .bat
```
🔧 KHẮC PHỤC:
1. Chuột phải file .bat → "Run as administrator"
2. Hoặc mở cmd, gõ: python main.py
```
